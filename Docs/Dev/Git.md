# Criando Repositorio no GitHub

Como criar um repositorio no github a partir de uma instalação:

1. Criar repositorio no github

2. Executar os seguintes comandos na pasta do jogo
```bash
echo "# SkyrimSE-DSilHand" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/AndersonPaschoalon/SkyrimSE-DSilHand.git
git push --set-upstream origin master
```

3. Criar .gitignore


4. Realizar push 
```bash
git add .gitignore
git commit -m "adicionando gitignore"
git push
```
